Gossip Tales

A escape-room-like game where you find yourself in the center of high school drama.

------
My friend, Vaish, and I, Evelyn, made this project for the Snoozefest event!

We made this project because it mainly uses HTML, CSS, and JS, which are very beginner-friendly coding languages.
We also want to try something unique, like this interactive website game.
We had so much fun making the storyline and writing the scripts in the game.
Credits to Vaish because she made practically wrote all of the scripts and did most of the storyline 😅
Coding this project was also very fun because we know how to code with HTML, CSS, and JS!

------
We primarily used HTML, CSS, and JS to create Gossip Tales.
We thought of Gossip Tales because we wanted to do something related the theme, Whisper, so we chose to center our project around gossip.
We mainly worked with functions like buttons, text, margins and padding, flexbox, etc.
We also made some modals.

------
We struggled a lot with pop-up modals. This part took the longest because Evelyn was introduced to modals not long ago
(and also brought it up to Vaish), but we haven't had to chance to fully understand it.
So, this project helped us learn more about making modals for future projects!
